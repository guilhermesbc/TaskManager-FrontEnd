# Task Manager App (Front-End)
Uma aplicação que possibilita a criação, edição e exclusão de tarefas organizacionais.

# Tecnologias Utilizadas:
.NET 8.0
.NET EntityFramework Core 8
MSSQL Server
Angular 13
Swagger API Document
